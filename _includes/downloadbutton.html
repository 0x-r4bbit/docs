The various pieces of {{site.project_title}} {{site.latest_version}} can be installed using Bower.

1. Install the polyfills 

bower install --save Polymer/platform

2. Install the core sugaring 

bower install --save Polymer/polymer

3. Install the elements 

bower install --save Polymer/polymer-elements
bower install --save Polymer/polymer-ui-elements

Note: --save adds the item to your app's bower.json as a dependency.
